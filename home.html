Blog Application

A simple blog application that allow users to submit posts about their day. The front end was designed using Semantic UI keeping all the blog posts in a list format. Readers can select “Read More” on the post that they are most interested in which will take them to a different page with more information on that blog post. The blog application uses Node.js, Express and MongoDB on the back end to route readers to the correct location as they browse the site. Following RESTful protocols, users that submit a post have full control to create, delete or edit their posts. Visit the site and try it out for yourself!

The Traveler's Guide

The Traveler’s Guide application is the ultimate guide to places you have to visit at some point in your life. Users submit breath-taking places they have been to share with others on the site. Traveler’s Guide features authorization and authentication with the use of the passport.js library. New users can sign up and upload their own posts. Existing users can log in to edit or delete their posts when they choose to. Users that have logged in can share their thoughts by commenting on other people’s post. Check out the site and upload your own destination!

A.I List Maker

This list maker application accepts photos from users to be analyzed. With the help of a machine learning API (Clarafai), items in the photos are used to create a list. Items that are missing from the initial analysis, can be added manually afterwards. This application was built using React on the front-end. Node.js, Express and Postgres was used to create the back-end API which would connect to the front-end of the application.
